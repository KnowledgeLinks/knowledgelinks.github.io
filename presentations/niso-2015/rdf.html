Resource Description Framework (RDF)

RDF is one method of implementing Linked Data.

What we can do with it?

  • Build RDF vocabularies to describe the how the information stored
  • Combine multiple uses into one source object
  • Provide linkage to data from other sources

RDF Vocabularies! How important are they?

Answer??? They are Very important…and Not as important as we make them out to be.

  • They are primary tool to allow for a machine readable description of the data
  • Not as important as we make them to be??? Why not?
    • They are a data tool
    • Therefore so long as we know what the vocabulary means we can easily translate from one vocabulary to another.

Challenge of moving from MARC to RDF
